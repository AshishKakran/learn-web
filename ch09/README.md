# HTML FORMS

## summary of form elements: 

**button**
 - type="submit|reset|button"
 - name="text"
 - value="text"
 - form,formaction,formmethod


 **datalist**

 **fieldset**
  - form="idvalue"
  - name="text"

  **form**
   - action="URL"
   - method="GET|POST"
   - name="text"

  **input**
   - type="submit|reset|button
   		   text|password|checkbox|radio|
   		   image|file|hidden|email|tel
   		   search|url|data|time|number|range|color"

  **legend**
   - for="text"

  **optgroup**
   - label="text"
   - disabled

   **option**
    - disabled
    - label
    - selected
    - value

   **output**
     - for
     - from
     - name

   **progress**

   **select**
     - autofocus
     - disabled
     - form
     - name
     - size

   **textarea**
     - autocomplete
     - autofocus
     - cols,rows
     - form
     - maxlength/minlength
     - name
     - placeholder
     - wrap="hard|soft"
